Breve História:
Podemos dizer que a história dos primeiros computadores começa com o Mark I (o primeiro a gerar cálculos avançados) em 1944.
Esses primeiros computadores ocupavam uma sala inteira e pesavam toneladas.
A segunda geração (1959-1965) marca a evolução de válvulas para transistores, isso diminui drasticamente o tamanho dos computadores.
De 1965 a 1970 temos a chegada dos CHIPS que deixaram ainda menores os computadores
Apartir de 1971 temos a 4° geração que é a atual, com a chegada dos computadores pessoais o uso se popularizou.
Dizem que já estamos na quinta geração. A controvérsias.

O que são Clients?

(voltar)

Clients (clientes):
São dispositivos ou programas que fazem solicitações a um servidor para acessar serviços, informações ou recursos.

Exemplos:
  • Um navegador (Chrome, Firefox, Edge) solicita páginas a um servidor web.
  • Um aplicativo de e-mail solicita novas mensagens ao servidor.
  • Um aplicativo de banco solicita dados ao servidor da instituição.
Em resumo: o cliente faz a requisição e o servidor responde.

O que são Servers?

(voltar)

Servers (servidores):
São computadores ou programas que fornecem serviços, dados ou recursos para outros computadores (clientes) por meio de uma rede, como a internet.

  • Web → entrega páginas HTML aos navegadores.
  • Servidor de Banco de Dados → armazena e fornece dados.
  • Servidor de Arquivos → disponibiliza arquivos para acesso.
  • Servidor de E-mail → envia e recebe mensagens.
Em resumo: o cliente faz uma solicitação, e o servidor processa essa solicitação e envia uma resposta.
